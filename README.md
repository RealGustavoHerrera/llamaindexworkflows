# LlamaIndex Workflow examples

This repository contains running examples of the workflow patterns explained in `https://developers.llamaindex.ai/python/llamaagents/workflows/`
This is for educational purposes only, none of the examples have any commercial value.

## 1) Basic Joke
Basic workflow with basic custom events

## 2) Branching workflows
Shows how to branch and loop a workflow execution

## 3) Concurrent workflows
Shows how to execute branchs concurrently and how to handle the outcome <i>(to wait or not to wait... that is the question)</i>

## 4) Custom Events
Shows how to create and utilize custom events (more advanced examples)

## 5) Drawing Flows
Shows how to visualize a flow (all possible paths or a given execution)

## 6) External state
Shows how to store state (in a dict) and recover it to continue later

## 7) Human in the loop
Shows how to ask for human input before continuing the execution of the workflow

## 8) Looping
Shows how to loop an execution (and how to put a limit to avoid infinite loops)

## 9) Multi-Agent Workflows
Shows how to use multiple specialized agents to perform a workflow. **Bonus track** includes telemetry (using https://app.arize.com/)

## 10) ReAct Agent
Shows how to create a `Reason and Act` agent. **Bonus track** includes telemetry (using https://app.arize.com/)

## 11) Streaming
Shows how to stream the results to the client

## 12) Using debugger
Shows how to use a workflow server (running in localhost) for debugging purposes
